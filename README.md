**DocxPro Studio** — это мощная среда разработки и автоматизации документов, позволяющая генерировать сложные DOCX и PDF отчеты с использованием JavaScript.

Проект разработан **СНН PROJECT** (Союз Независимых Наработок).

---

## 🚀 Возможности

*   **Генерация документов:** Создание DOCX файлов любой сложности с помощью JS (на базе библиотеки `docx`).
*   **Конвертация в PDF:** Встроенный конвертер (использует MS Word API).
*   **Песочница (Sandbox):** Безопасное исполнение пользовательского кода.
*   **Auto-Fix:** Автоматическое исправление ошибок в коде и импортах.
*   **Умный редактор:** Подсветка синтаксиса, автодополнение и валидация ошибок.
*   **Локальный сервер:** Работает как десктопное приложение (EXE) с веб-интерфейсом.

## 🛠 Установка и Запуск (для разработчиков)

### Требования
*   Node.js v18+
*   Microsoft Word (для конвертации в PDF и диагностики)

### Сборка из исходников

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/HermanDebush/DocxPro-Studio.git
    cd DocxPro-Studio
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```

3.  **Запустите в режиме разработки:**
    ```bash
    node index.js
    ```

4.  **Соберите EXE файл (Windows):**
    ```bash
    npm install -g pkg
    pkg . --targets node18-win-x64 --output docxpro-win.exe
    ```

## 📂 Структура Проекта

*   `src/` — Исходный код бэкенда (сервер, песочница, конвертер).
*   `public/` — Фронтенд (HTML, CSS, JS).
*   `index.js` — Точка входа (запуск сервера и браузера).
